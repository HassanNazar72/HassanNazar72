### 🚀 Featured Project — SnapLink

> **Serverless screenshot sharing platform built entirely on AWS**

Upload a screenshot → get a short shareable link. No servers. No idle cost. Just AWS services wired together the right way.

| What | How |
|---|---|
| Auth | Google OAuth 2.0 via Amazon Cognito (JWT, 24h tokens) |
| Upload | Presigned S3 PUT — file bytes never touch Lambda |
| Short links | API Gateway → Lambda → DynamoDB lookup → 302 redirect |
| CDN | CloudFront + Origin Access Control (S3 never publicly exposed) |
| Abuse control | Per-user quota enforced server-side via DynamoDB GSI COUNT |
| IaC | Entire stack in one SAM/CloudFormation template |

**Stack:** `AWS Lambda` `API Gateway` `S3` `DynamoDB` `CloudFront` `Cognito` `Node.js 20` `SAM`
